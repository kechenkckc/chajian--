# 蒲公英达人导出同步飞书插件

这是一个 Chrome MV3 扩展，用于在蒲公英页面导出当前筛选条件下的达人，并同步到飞书电子表格。

## 功能

- 导出当前达人：按蒲公英当前筛选条件导出达人 CSV。
- 导出后直写飞书：勾选后，导出完成会直接写入同步达人飞书表格。
- 同步达人到飞书：先保存配置，再检测飞书表格可写，最后同步当前筛选或已下载 CSV。
- 补足详情：读取另一张“需补足详情”的飞书表格，逐个打开达人详情页补足字段和截图。
- 多子表补足详情：检测到详情表包含多个子表时，可开启“遍历所有子表”，每个子表会按自己的表头格式单独识别和写回。

## 飞书配置

同步达人和补足详情共用：

- 飞书 App ID
- 飞书 App Secret

同步达人使用：

- 同步达人飞书链接
- 同步子表 ID，可留空

补足详情使用：

- 需补足详情的飞书表格链接
- 详情子表 ID，可留空
- 检测到多子表时遍历所有子表，可选

注意：补足详情的飞书链接不能和同步达人飞书链接相同。
如果填写了详情子表 ID，则只补足该子表；如果未填写且检测到多个子表，可勾选遍历所有子表。

飞书应用至少需要开通电子表格或云空间的读写权限，例如：

- `sheets:spreadsheet`
- `sheets:spreadsheet:read`
- `sheets:spreadsheet:readonly`
- `drive:drive`
- `drive:drive:read`
- `drive:drive:readonly`

## 使用流程

1. 在 Chrome 扩展管理页加载本目录。
2. 打开蒲公英达人列表页并完成筛选。
3. 打开扩展侧边栏。
4. 先填写并保存飞书配置。
5. 使用“导出当前达人”“同步达人到飞书”或“补足详情”。

## 业务规则

- 未填写 App ID、App Secret、同步达人飞书链接时，“导出后直接写回飞书”不能勾选。
- “同步达人到飞书”会先保存配置，再检测是否能写入，检测通过后才开始同步。
- “已下载达人表格同步”已合并到“同步达人到飞书”，通过来源切换选择 CSV。
- “补足详情”会先检测详情表。如果发现多个子表，用户可以勾选“遍历所有子表”；各子表按自己的表头结构识别缺失字段，不要求所有子表格式一致。
- 预留 API 功能已取消。
